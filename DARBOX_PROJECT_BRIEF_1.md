### Текущие фичи бота (v4):
1. **Анкета 20 вопросов** (ольфакторный портрет):
   - Пол (с уточнением М/Ж для подарка), возраст, образ жизни
   - Повод, интенсивность, парфюмерный опыт
   - Любимые/нелюбимые ноты (18 направлений, мульти-выбор)
   - Текущие ароматы, сезон, время суток, настроение
   - Ассоциации (12 вариантов), стойкость, эксперименты
   - Бюджет, гардероб, аллергии, цель подписки, пожелания
2. **Личный кабинет** — профиль с ольфакторным портретом и статистикой
3. **Парфюмерный дневник** — записи о любых ароматах с тегами настроения
4. **Оценки (3 параметра)** — общее впечатление + стойкость + шлейф (1-5)
5. **Карта вкуса** — визуальная аналитика на основе оценок, топ фавориты
6. **Реферальная система** — персональная ссылка, 10% за друга
7. **Навигация** — главное меню из 9 разделов, кнопка "← Назад" везде
8. **Чат** — клиент ↔ админ через бота, всё сохраняется в БД
9. **Доставка** — выбор Почта/СДЭК/Курьер с ценами, ФИО, город, адрес, индекс
10. **Напоминания** — /remind за 3 дня до доставки (подтвердить/сменить адрес)
11. **Админ-панель** — /admin, /subs, /profile, /reviews, /chat, /due, /broadcast

### БД (SQLite):
- users (все поля анкеты + подписка + доставка + статус + paid_at + ref)
- feedback (user_id, month, aroma, rating_overall/longevity/sillage, comment)
- diary (user_id, aroma, text, mood, occasion, rating)
- messages (user_id, direction, text)
- box_history (user_id, month, aromas, tracking)
